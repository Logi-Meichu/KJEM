Ajuda do botão de gesto

Gestos

Com os gestos, você tem à sua disposição uma forma fácil e intuitiva de interagir com o seu sistema operacional e seus aplicativos.

Use os gestos para aumentar ou diminuir o zoom em documentos, navegar por páginas da Web, girar imagens e muito mais.

O Logitech Options permite que você use diversos gestos com mouses Logitech compatíveis.

Ativar conjuntos de gestos

Clique na guia Mouse e, em seguida, clique em qualquer um dos botões realçados.

Quando o botão de gesto estiver selecionado na lista de botões ativos, os conjuntos de gestos disponíveis será exibido à direita, juntamente com um guia visual do conjunto selecionado.

Clique para selecionar o botão do conjunto de gestos que deseja habilitar. O conjunto selecionado permanece ativo até que você selecione outro conjunto.

Nota: Você pode atribuir a ação de gesto a outros botões.   Os gestos permanecerão indisponíveis se a ação de gestos não estiver atribuída a um botão.


Realizar gestos

Para realizar um gesto, mantenha o botão de gestos pressionado enquanto move o mouse para a direita, para a esquerda, para cima ou para baixo. Clique no botão de gesto sem mover o mouse para realizar a ação relacionada.
