Использование кнопки жестов

Жесты

С помощью жестов можно интуитивно и легко управлять операционной системой и приложениями.

Жесты позволяют увеличивать масштаб отображения документов, просматривать веб-страницы, поворачивать изображения и многое другое.

ПО Logitech Options позволяет выполнять целый ряд жестов с помощью совместимых мышей Logitech.

Включение наборов жестов

Откройте вкладку Мышь и нажмите любую выделенную кнопку жестов.

Когда кнопка жестов выбрана в списке действий кнопок, справа отображаются наборы жестов вместе с иллюстрацией выбранного набора.

Нажмите, чтобы выбрать кнопку, которой необходимо присвоить набор жестов. Выбранный набор остается активным, пока вы не выберете другой.

Примечание. Вы также можете назначить действие жеста другим кнопкам.   Если действие жеста не назначено ни одной кнопке, жесты недоступны.


Выполнение жестов

Чтобы выполнить жест, переместите мышь влево, вправо, вверх или вниз, удерживая кнопку жестов. Для выполнения связанного действия нажмите кнопку жестов, удерживая мышь неподвижной.
