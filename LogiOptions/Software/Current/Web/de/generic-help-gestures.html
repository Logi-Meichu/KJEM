Hilfe zum Gestenschalter

Gesten

Gesten sind eine intuitive, einfache Möglichkeit, um mit Ihrem Betriebssystem und Ihren Anwendungen zu interagieren.

Verwenden Sie Gesten um Dokumente zu vergrößern, Webseiten zu durchsuchen, Bilder zu drehen und viele weitere Aktionen.

Mit Logitech Options können Sie eine Vielzahl von Gesten mit Ihrer kompatiblen Logitech Maus ausführen.

Gestensets aktivieren

Öffnen Sie die Registerkarte Maus und klicken Sie auf einen beliebigen markierten Schalter.

Wenn der Gestenschalter in der Liste von Tastenaktionen ausgewählt ist, wird das Gestenset auf der rechten Seite zusammen mit einer visuellen Anleitung für das ausgewählte Set angezeigt.

Klicken Sie hier, um das Gestenset auszuwählen, das Sie aktivieren möchten. Das ausgewählte Set bleibt aktiv, bis ein anderes ausgewählt wird.

Hinweis: Sie können den Gestenschalter auch anderen Tasten zuweisen.   Gesten sind nicht verfügbar, wenn die Gestenaktion keiner Taste zugewiesen ist.


Gesten ausführen

Um eine Geste auszuführen, bewegen Sie bei gedrücktem Gestenschalter die Maus nach links, rechts, oben oder unten. Klicken Sie auf den Gestenschalter, ohne die Maus zu bewegen, um die entsprechende Aktion auszuführen.
