제스처 버튼 도움말

제스처

제스처는 운영 체제 및 응용 프로그램과 간편하고 손쉽게 상호작용하는 방법을 제공합니다.

문서를 확대하거나 웹 페이지 검색, 이미지 회전 등에서 제스처를 사용할 수 있습니다.

Logitech Options를 이용하면 호환 가능한 Logitech 마우스로 다양한 제스처를 수행할 수 있습니다.

제스처 세트 활성화

마우스 탭을 클릭한 다음, 강조 표시된 버튼 중 하나를 클릭합니다.

버튼 동작 목록에서 제스처 버튼이 선택된 경우, 선택한 세트에 대한 시각적 가이드와 함께 오른쪽에 사용 가능한 제스처 세트가 나타납니다.

활성화할 제스처 세트에 대한 버튼을 클릭해서 선택합니다. 다른 세트를 선택할 때까지 선택한 세트가 활성 상태로 유지됩니다.

참고: 제스처 동작을 다른 버튼에 대신 할당할 수 있습니다.   제스처 동작이 아무 버튼에도 지정되지 않은 경우, 제스처를 사용할 수 없습니다.


제스처 수행

제스처를 수행하려면 제스처 버튼을 누른 상태에서 마우스를 왼쪽, 오른쪽, 위 또는 아래로 이동합니다. 마우스를 움직이지 않고 관련 동작을 수행하려면 제스처 버튼을 클릭하십시오.
