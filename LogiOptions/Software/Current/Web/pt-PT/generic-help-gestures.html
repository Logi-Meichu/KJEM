Ajuda do Botão de gestos

Gestos

Os gestos oferecem um modo intuitivo e fácil para interagir com o seu sistema operativo e aplicações.

Utilize os gestos para ampliar o zoom nos documentos, navegar em páginas Web, rodar imagens e muito mais.

O Logitech Options permite-lhe efectuar uma série de gestos com ratos Logitech compatíveis.

Activar conjuntos de gestos

Clique no separador Rato e clique num dos botões realçados.

Quando o botão de gestos se encontra seleccionado na lista de acções do botão, os conjuntos de gestos disponíveis são apresentados à direita, juntamente com um guia visual do conjunto seleccionado.

Clique para seleccionar o botão para o conjunto de gestos que pretende activar. O conjunto seleccionado permanece activo até seleccionar outro conjunto.

Nota: Pode, ao invés, atribuir a acção de gestos a outros botões.   Se a acção de gestos não estiver atribuída a nenhum botão, os gestos não estão disponíveis.


Efectuar gestos

Para efectuar um gesto, mantenha premido o botão de gestos enquanto desloca o rato para a esquerda, para a direita, para cima ou para baixo. Clique no botão de gestos sem mover o rato para executar a acção relacionada.
