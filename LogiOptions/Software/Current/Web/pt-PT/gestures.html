Ajuda do MX Master

Gestos

Os gestos oferecem um modo intuitivo e fácil para interagir com o seu sistema operativo e aplicações.

Utilize os gestos para ampliar o zoom nos documentos, navegar em páginas Web, rodar imagens e muito mais.

O software Logitech Options permite o acesso a vários gestos que pode realizar com o rato MX Master.

Activar conjuntos de gestos

Abra o separador Rato e clique no botão de gestos realçado.

Quando o botão de gestos estiver seleccionado na lista de acções do botão, os conjuntos de gestos aparecem à direita juntamente com um guia visual do conjunto seleccionado.

Um conjunto seleccionado permanece activo até que seleccione outro conjunto.

Nota: Os gestos estão indisponíveis se a acção do gesto não estiver atribuída. (Também pode atribuir o botão de gestos ao botão do polegar, botão central e botão de mudança de modo.)


Efectuar gestos

Para efectuar um gesto, mantenha premido o botão de gestos enquanto desloca o rato para a esquerda, para a direita, para cima ou para baixo.

Se clicar no botão de gestos, efectuará a acção relacionada com os gestos activos.



Para obter mais informações, consulte o Guia de Imersão do MX Master.
