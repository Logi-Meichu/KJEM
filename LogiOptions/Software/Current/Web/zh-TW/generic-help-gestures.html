手勢按鈕說明

手勢

手勢提供與作業系統和應用程式的簡單直覺互動方式。

使用手勢可進行放大顯示文件、瀏覽網頁、旋轉影像等等動作。

Logitech Options 讓您可使用相容的 Logitech 滑鼠執行各種不同手勢。

啟用手勢設定

按一下「滑鼠」標籤，然後按一下任一個反白的按鈕。

當在按鈕動作清單中選取手勢按鈕後，可用的手勢設定會出現在右方，並提供有關於選取之設定的視覺指南。

按一下可選擇要啟用之手勢設定的按鈕。 除非選取不同的設定，否則選取的設定會一直保持有效。

注意：您可以將手勢動作改為指定到其他按鈕。   如果手勢動作沒有指派給任何按鈕，就無法使用手勢。


執行手勢

若要執行手勢，請在上下左右移動滑鼠時，按住手勢按鈕不放。 按一下手勢按鈕無需移動滑鼠即可執行相關動作。
